Gute Gründe, nicht den Internet Explorer zu benutzen

  1. Der IE ist unsicher.
    Es gibt Unmengen von Sicherheitslücken, die die Sicherheit Ihres Computers und damit Ihrer Daten stark gefährden!
  2. Von Haus aus eingebaute Funktion, die Daten über Ihre Surfgewohnheiten an die Firma "Alexa" sendet.
  3. Sie können im IE ohne Zusatzsoftware keine Umlautdomains wie schröder.de oder müller.de aufrufen.
  4. Mit der Benutzung des IE unterstützen Sie eine Monopol-Gesellschaft die im Endeffekt nur darauf aus ist, Ihr Geld zu kriegen.
  5. Microsoft gab sogar zu, dass es keine neuen alleinstehenden Versionen des IE mehr geben wird - wer also künftig den aktuellsten IE haben möchte, wird gezwungen sein, auch sein Windows zu aktualisieren.
  6. Obwohl das moderne Grafikformat PNG, das eine Fülle von Featiures, die gerade auch für Webseiten interessant sind, bereits seit 1997 existiert, ist der Internet Explorer heute - 7 Jahre (!!) später - noch immer nicht in der Lage eines der Hauptneuerungen von PNG gegenüber den älteren Formaten JPEG und GIF - nämlich die mehrstufige Transparenz in 24-Bit-Dateien ("Alpha-Channel") korrekt anzuzeigen. Hintergründe erscheinen nicht transparent sondern grau!
  7. Weiterhin unterstützt der IE einige Features der modernen Webstandards XHTML und CSS nicht: Das W3C-Event-Modell und die fixe Positionierung von Elementen per CSS seien als Beispiele genannt.
  8. Wenn ein Website (ungünstigerweise) eine spezifische Eigenschaft/Funktion des Internet Explorers benutzen möchte, sind nur Benutzer mit Windows und dem IE in der Lage die Website korrekt zu benutzen. Linux, Mac & Co bleiben außen vor.
  9. Microsoft wird keine neuen Versionen des IE mehr für den Mac rausbringen.

 

Bessere Gründe, Mozilla Firefox zu benutzen

  1. Keine Spionage-Funktionen
  2. Eingebauter Popup-Blocker
  3. Zeigt Seiten schnell an
  4. Eingebauter Mini-Download-Manager mit "Download unterbrechen+fortsetzen"-Funktion
  5. Leicht zu bedienen, auch für Einsteiger
  6. Sie können Umlautdomains ohne Zusatzsoftware aufrufen.
  7. 95%+ aller Internet-Seiten funktionieren mit Firefox genausogut wie mit dem IE.
  8. Auf 100% aller Internet-Seiten kann man die Textgröße anpassen - während es für einen Webdesigner kein Problem ist, das im IE zu verhindern.
  9. Browsen mit mehreren Tabs, also bequemes Umschalten zwischen verschiedenen Webseiten ohne überfüllte Taskleiste.
  10. Es stehen sehr viele, leistungsfähige, kostenlose Plug-Ins für den Firefox zur Verfügung.
  11. Eingebaute Such-Toolbar für Google und jede andere Suchmaschine
  12. Mit einem Mausklick mit der rechten Maustaste auf einen markierten Text kann man sofort mit Google nach diesem Text suchen.
  13. Mit der Funktion "Seiteninformationen anzeigen" im Kontextmenü können viele Daten über die Seite angezeigt werden, u.a. auch versteckte Formularfelder.
  14. Das Programm ist kostenlos und "Open Source".
  15. Besser als sein "Verwandter" Netscape
